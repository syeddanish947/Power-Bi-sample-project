A clean and interactive Power BI dashboard built to analyze e-commerce sales data.

1. Project Title
E-Commerce Sales Insights Dashboard
2. Short Description / Purpose

This dashboard provides a simple and clear view of e-commerce sales performance. It helps users understand revenue, orders, profit, and product/category trends through easy-to-read visuals and interactive filters.

3. What I Did (Process Overview)

I followed a basic Power BI workflow without using any DAX:

Uploaded the raw sales data

Cleaned and modified the data using Power Query (Transform Data panel)

Created relationships between tables in the Model View

Designed the dashboard in the Report View

Added charts, KPIs, and slicers to make the report easy to explore

Ensured that slicers worked smoothly across all visuals

Analyzed sales clearly based on product, customer, region, and category information

4. Tech Stack

Power BI Desktop

Power Query (for cleaning and transforming data)

Data Modeling (creating relationships)

5. Features / Highlights

Simple and attractive dashboard design

Interactive slicers for filtering by category, region, customer segment, etc.

Visuals like bar charts, line charts, cards, and tables

Easy comparison of sales across products and regions

Clear understanding of total sales, orders, and profit from the dataset
